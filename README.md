# 💳 YourBank API

YourBank API is a secure and scalable backend banking system built with **Spring Boot**.  
It provides RESTful APIs for managing user accounts, performing deposits, withdrawals, transfers,  
and retrieving transaction history — all secured with **JWT authentication**.

---

## 🚀 Features
- **User Authentication** – Sign up, login, and JWT-based security.
- **Account Management** – Create and manage bank accounts.
- **Transactions** – Deposit, withdraw, and transfer funds.
- **History Tracking** – View transaction history and account balance.
- **MySQL Database** – Reliable and persistent data storage.

---

## 🛠 Tech Stack
- **Backend:** Spring Boot, Spring Security, Spring Data JPA
- **Database:** MySQL
- **Authentication:** JWT (JSON Web Tokens)
- **Build Tool:** Maven
